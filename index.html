HTML

The basic structure of an HTML page:

Basic selectors

Tag type selector: p { ... }

Class selector: .some-class { ... }

ID selector: #some-id { ... }

Attribute selector (new): [src] { ... }

Universal selector (new): * { ... }

Grouping selectors

Grouping selector / selector list: p, .some-class { ... }

Combinators

Combined selector: p.some-class { ... }

Descendant combinator: div span

Child combinator: ul > li

General sibling combinator: p ~ span

Adjacent sibling combinator: h2 + p

Pseudo-classes and pseudo-elements

Pseudo element: a:hover { ... }

Pseudo class: p::first-line { ... }
